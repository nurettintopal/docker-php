## usage

build image:
```sh
$ docker build -t docker-php .
```

run container:
```sh
$ docker run -d -p 8080:8080 docker-php
```

open a browser and go to:
```sh
http://localhost:8080/
```

connect to container with ssh:
```sh
$ docker exec -it CONTAINER_ID sh
```

delete container:
```sh
$ docker rm -f CONTAINER_ID
```
